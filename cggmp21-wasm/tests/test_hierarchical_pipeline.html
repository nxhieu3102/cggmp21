🧠 Hierarchical Threshold Concept

Key Innovation: Unlike traditional threshold schemes where any t out of n parties can recover the secret, hierarchical threshold schemes assign ranks that determine access privileges.

  • Rank 0: Highest privilege (Parties 0, 1)
  • Rank 1: Standard privilege (Parties 2, 3)

Recovery Rule: For any subset of t=3 parties with ranks (r₁, r₂, r₃) sorted in ascending order, secret recovery is possible if and only if rᵢ ≤ i-1 for all i ∈ {1, 2, 3}.
